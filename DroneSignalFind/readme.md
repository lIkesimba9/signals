## Консольный обнаружитель сигнала

Параметры обнаружителя хранятся в файле params.txt. Путь к нему должен передаваться в функции обнаружителя.
prefix_length - длина префикса кадра
data_length - длина данных в кадре
frames_sum - количество кадров накопления корреляции
corr_burst - порог обнаружения сигнала
find_flag_path find_flag - путь к файлу-флагу обнаружения сигнала (без пробелов и кириллицы)
correlation_path corr.txt - путь к файлу с числовыми данными коррелятора (без пробелов и кириллицы)
correlation_file_max_size - максимальный размер файла с данными коррелятора
channel - номер канала (0 или 1)

Файл params.txt может правиться сторонним приложением (управляющим веб-приложением)

### Обнаружение сигнала из данных, передаваемых через буфер
Фоновый процесс, записывающий сигнал в буфер, после заполнения буфера должен дергать функцию signal_frame_find_buf

### Обнаружение сигнала из данных, передаваемых через файл
Процесс, записывающий сигнал в файл, после записи и закрытия файла должен дергать функцию signal_frame_find_file
